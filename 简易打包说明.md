# 💡 简易打包方案（不需要管理员权限）

由于 Electron 打包需要管理员权限创建符号链接，这里提供一个更简单的替代方案。

---

## 🎯 推荐方案：使用"一键启动"脚本

### 给用户的使用方式：

1. 打包整个项目文件夹（不用 electron-builder）
2. 用户下载后解压
3. 双击"一键启动.bat"即可使用

### 优点：
- ✅ 无需管理员权限
- ✅ 打包简单快速
- ✅ 用户体验一样好
- ✅ 文件更小（不包含 Electron）

### 缺点：
- ❌ 用户需要安装 Node.js（但有自动检测和提示）

---

## 📦 打包步骤

### 第1步：准备发布包

```bash
# 1. 构建生产版本
npm run build

# 2. 创建发布文件夹
mkdir release
```

### 第2步：复制必要文件

复制以下文件到 `release` 文件夹：
- `一键启动.bat`
- `.next/`
- `app/`
- `public/`
- `node_modules/`
- `package.json`
- `next.config.ts`
- `README.md`

### 第3步：压缩打包

```bash
# 压缩成 zip 文件
Compress-Archive -Path release\* -DestinationPath "角色卡设计器-v1.0.0.zip"
```

### 第4步：上传分享

将 `角色卡设计器-v1.0.0.zip` 上传到：
- GitHub Releases
- 百度网盘
- 阿里云盘

---

## 📖 用户使用指南

**下载后的使用方法**：

```
1. 下载并解压 "角色卡设计器-v1.0.0.zip"
2. 双击 "一键启动.bat"
3. 如果没有 Node.js，会自动提示下载安装
4. 安装后重新双击"一键启动.bat"
5. 等待3-5秒，自动打开界面
6. 开始使用！
```

---

## 🚀 如果仍想使用 Electron 打包

**需要以管理员权限运行**：

1. 右键点击 PowerShell
2. 选择"以管理员身份运行"
3. 执行：

```powershell
cd "E:\ev\新建文件夹 (2)\bmad-writing-project\psyche-card-designer"
npm run dist:portable
```

等待打包完成（约5-10分钟），在 `dist/` 文件夹获取 `.exe` 文件。

---

## 对比

| 方案 | 文件大小 | 需要权限 | 用户需要 Node.js |
|------|---------|---------|-----------------|
| **简易打包** | 约 50-100 MB | ❌ 不需要 | ✅ 需要（有自动提示） |
| **Electron** | 约 200-300 MB | ✅ 需要管理员 | ❌ 不需要 |

---

**建议**：先用简易打包方式发布，如果用户反馈需要免 Node.js 版本，再考虑 Electron 打包。
