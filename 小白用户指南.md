# 🎯 小白用户指南 - 3步开始使用

**不懂编程？没关系！** 跟着这个教程，3步就能用上角色卡设计器！

---

## 📥 方式1：下载便携版（最简单）⭐

### 第1步：下载

点击下载链接，获取便携版应用：

**👉 [下载：角色卡设计器-便携版.exe](https://github.com/baixiong443/character-card-designer/releases)**

> 💡 提示：选择最新版本，下载以 `.exe` 结尾的文件

### 第2步：运行

1. 下载完成后，找到 `.exe` 文件
2. **双击运行**
3. 等待 3-5 秒，程序会自动打开

> ⚠️ Windows 可能提示"无法验证发布者"，点击"仍要运行"即可

### 第3步：配置 AI

首次使用需要配置 AI：

1. 点击右上角 **⚙️ 设置**
2. 填写你的 AI 配置：
   - **API 密钥**：你的 OpenAI/Claude API Key
   - **API 端点**：默认即可
   - **模型**：建议选 `gpt-4`
3. 点击 **测试连接**
4. 看到"✅ 连接成功"后，点击**保存**

**🎉 完成！现在可以开始创作角色卡了！**

## 📱 方式3：手机使用

### 在电脑上运行服务

如果你下载了便携版，手机也能用！

1. 在电脑上双击运行 `.exe`
2. 查看电脑 IP 地址：
   - Windows：打开 CMD，输入 `ipconfig`
   - 找到 IPv4 地址，例如：`192.168.1.100`
3. 手机浏览器打开：`http://你的IP:3000`
   - 例如：`http://192.168.1.100:3000`

> 💡 提示：手机和电脑要在同一个 WiFi 下

## ⚙️ 获取 API 密钥

### OpenAI API Key

1. 访问：https://platform.openai.com/api-keys
2. 登录/注册账号
3. 点击 **Create new secret key**
4. 复制密钥（以 `sk-` 开头）

### Claude API Key

1. 访问：https://console.anthropic.com/
2. 登录/注册账号
3. 进入 **API Keys** 页面
4. 创建并复制密钥

### 其他 API

支持所有兼容 OpenAI 格式的 API：
- 本地模型（Ollama、LM Studio）
- 中转 API
- 自建服务

## 📖 使用教程

### 创建第一个角色卡

1. **阶段0：导入文本**（可选）
   - 如果有小说/文档，可以粘贴进来
   - AI 会自动分析角色信息
   - 没有文本也可以跳过

2. **阶段1-8：逐步完善**
   - 点击"开始生成"，AI 自动创建
   - 可以手动编辑每个阶段
   - 点击"💬 继续对话"与 AI 交流

3. **阶段9：最终整合**
   - 分9步生成完整角色卡
   - 自动合并所有信息
   - 确保内容完整无遗漏

4. **导出角色卡**
   - 点击"导出"按钮
   - 选择格式：JSON/PNG/Markdown
   - 保存到本地

### 导入到 SillyTavern

1. 导出为 `JSON V3` 格式
2. 打开 SillyTavern
3. 点击"导入角色"
4. 选择导出的 `.json` 文件
5. **完成！**

## ❓ 常见问题

### Q1: 双击 .exe 没反应？

**A**: 
- 等待 3-5 秒，程序需要启动时间
- 检查是否被杀毒软件拦截
- 尝试"以管理员身份运行"

### Q2: 提示"连接失败"？

**A**: 
- 检查 API 密钥是否正确
- 检查网络连接
- 检查 API 额度是否充足

### Q3: 生成的角色卡不完整？

**A**: 
- 检查 AI 模型是否选对（建议用 gpt-4）
- 检查 maxTokens 设置（建议 16000）
- 尝试手动补充缺失内容

### Q4: 如何保存进度？

**A**: 
- 应用会自动保存进度
- 关闭后重新打开，进度还在
- 也可以点击"导出"备份

### Q5: 如何创建多个项目？

**A**: 
- 点击左上角项目名称
- 选择"新建项目"
- 输入项目名称即可
