# ⚡ 快速开始指南

> 5分钟上手 Psyche 角色卡设计器

---

## 🚀 第一步：安装

```bash
# 1. 进入项目目录
cd psyche-card-designer

# 2. 安装依赖（首次运行需要）
npm install

# 3. 启动应用
npm run dev
```

**打开浏览器访问**: http://localhost:3000

---

## ⚙️ 第二步：配置 AI

1. 点击右上角**齿轮图标** ⚙️
2. 填写 AI 设置

### 推荐配置：本地模型（免费）

```
AI 提供商: 自定义
API 端点: http://localhost:11434/v1/chat/completions
API 密钥: (留空)
模型: llama2
温度: 0.8
最大 Token: 8000
```

**前提**: 需要安装 [Ollama](https://ollama.ai)

### 或使用 OpenAI

```
AI 提供商: OpenAI
API 端点: https://api.openai.com/v1/chat/completions
API 密钥: sk-你的密钥
模型: gpt-4
温度: 0.8
最大 Token: 8000
```

3. 点击**测试连接**确认设置正确
4. 点击**保存**

---

## 🎨 第三步：创建角色

### 方式 A：AI 工作流（推荐新手）

1. 点击"**AI 工作流**"标签
2. 按照 8 个阶段依次创作：
   - **阶段1**: 基础身份（名字、物种、性格）
   - **阶段2**: 深度背景（故事、动机）
   - **阶段3**: 物理描写（外貌、特征）
   - **阶段4**: 专项深化（选择类型：Furry/BL/RPG等）
   - **阶段5**: 互动设计（对话风格、示例）
   - **阶段6**: 世界书（场景、触发词）
   - **阶段7**: 质量检查
   - **阶段8**: 导出

3. 每个阶段：
   - 输入你的想法
   - AI 会提供建议和选项
   - 满意后点击"**下一阶段**"

### 方式 B：直接编辑（适合有经验的用户）

1. 点击"**角色编辑器**"标签
2. 手动填写所有字段
3. 点击"**导出**"选择格式

---

## 📦 第四步：导出角色卡

1. 完成创作后，点击"**导出**"
2. 选择格式：
   - **JSON V3**（推荐，SillyTavern原生支持）
   - **PNG Card**（图片嵌入，拖拽导入）
   - **Markdown**（人类可读）
3. 下载到本地

---

## 🎮 第五步：使用角色卡

### 在 SillyTavern 中导入

1. 打开 SillyTavern
2. 进入"角色"页面
3. 拖拽你导出的 **PNG Card** 或 **JSON** 文件
4. 开始聊天！

## 💡 常用技巧

### 技巧 1：使用专项深化

在**阶段4**，根据角色类型选择合适的专项：
- **Furry 角色** → 选择"Furry兽人"
- **BL 角色** → 选择"BL/耽美"
- **游戏角色** → 选择"RPG/游戏"
- **日常角色** → 选择"跳过"

### 技巧 2：导入现有角色卡

1. 点击"导入现有角色卡"
2. 选择 JSON 文件
3. AI 会自动解析并分配到各阶段
4. 继续编辑或直接导出

### 技巧 3：使用世界书

在**阶段6**：
- 添加世界书条目（地点、人物、物品）
- 设置触发关键词
- 可以使用"AI生成"自动创建

### 技巧 4：自定义 AI 提示词

如果需要破限或更重口味：
1. 设置 → "自定义 System Prompt"
2. 勾选"启用"
3. 点击"加载模板"
4. 根据需求修改

## ❓ 遇到问题？

### AI 无法连接

- 检查 API 端点是否正确
- 检查 API 密钥是否有效
- 点击"测试连接"查看错误信息

### 生成内容不满意

- 调整温度（0.8-1.2）
- 增加 Max Tokens（8000-16000）
- 启用自定义提示词

### 手机无法访问

- 确保手机和电脑在同一 WiFi
- 检查防火墙设置
- 尝试 `0.0.0.0:3000` 而不是 `localhost:3000`
